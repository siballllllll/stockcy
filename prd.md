📄 [PRD] AI 단타 트레이딩 어시스턴트: 스톡시 (Stockcy)
1. 프로젝트 개요
프로젝트명: 스톡시 (Stockcy)

서비스 한 줄 소개: 실시간 뉴스 분석과 글로벌 수급 데이터를 바탕으로 최적의 단타 타점을 잡아주는 나만의 AI 트레이딩 비서

핵심 목표:

감정에 휘둘리지 않는 데이터 기반의 기계적 매매 환경 구축

실시간 뉴스 및 수급(매수/매도량) 분석을 통한 단타 승률 극대화

명확한 매수가, 매도가(익절/손절) 및 AI 분석 근거 제시

2. 사용자 시나리오
시장 선택: 사용자는 앱 접속 후 상단 토글을 통해 [국내 주식 🇰🇷 / 미국 주식 🇺🇸] 중 하나를 선택한다.

데일리 브리핑: 접속 시 나타나는 모달창을 통해 전날 핵심 뉴스 요약과 오늘 주목할 '주도 섹터'를 확인한다.

추천주 탐색: 브리핑 창을 닫고 메인 탭에서 실시간 단타 추천 종목 리스트(테이블 형태)를 확인한다. 이때 수급이 겹치는 '중복 섹터' 종목은 시각적으로 강조된다.

심층 분석: 특정 종목을 클릭하여 AI가 작성한 리포트(근거, 분석, 리스크)를 읽고 타임프레임(5/15/30분 봉)별 흐름을 파악한다.

실제 매매 및 기록: 분석 결과를 바탕으로 외부 앱(토스, 키움 등)에서 매매를 진행하며, 앱은 추천 타점 도달 여부를 자동으로 트래킹하여 수익률 데이터를 쌓는다.

3. 핵심 기능 목록
듀얼 마켓 모니터링: 🇰🇷/🇺🇸 시장 스위칭 및 실시간 데이터 연동

AI 데일리 브리핑: 장 시작 전/장 중 글로벌 뉴스 요약 및 수혜 섹터 매칭

AI 단타 추천 엔진: 매수가, 익절가, 손절가 자동 산출 및 추천 근거 요약

지능형 섹터 그룹핑: 동일 테마 종목 그룹화 및 강한 수급이 겹치는 종목 하이라이트

성과 자동 기록 시스템: 추천 타점 도달 여부 실시간 모니터링 및 누적 승률/수익률 계산

4. 기술 스택
Language & Framework: Python, Streamlit

Stock Data API: 한국투자증권 Open API (국내), yfinance (미국)

AI Engine: OpenAI GPT-4o 또는 Anthropic Claude 3.5 Sonnet

Database: Google Sheets (gspread 또는 st.connection 활용)

Hosting: Streamlit Community Cloud (무료 호스팅 환경)

5. 화면 구성
상단 영역: 시장 선택 토글, 데일리 브리핑 모달

탭 1 (실시간 타점 보드):

주도 섹터 전광판

실시간 추천 종목 테이블 (종목명, 현재가, 타점, 상태 등)

AI 인사이트 리포트 (종목 클릭 시 하단/사이드에 노출)

탭 2 (성과 트래킹): 누적 승률 대시보드 및 추천 히스토리 테이블

6. 상세 기능 명세
멀티 타임프레임 분석: 5분/15분/30분 봉 데이터를 종합하여 단기 추세와 세력 수급을 복합적으로 판단.

AI 리포트 자동 생성: 클릭 시점에 즉시 [뉴스 + 지표 + 수급]을 결합한 리포트와 3줄 요약 제공.

구글 시트 연동: 추천 발생 시점의 데이터와 결과(성공/실패)를 구글 시트에 행(Row) 단위로 자동 기록.

7. 디자인 가이드
테마: 다크 모드(Dark Mode) 기반의 프로페셔널 UI

컬러 시스템 (로컬라이징):

국내 주식: 상승(Red 🔴) / 하락(Blue 🔵)

미국 주식: 상승(Green 🟢) / 하락(Red 🔴)

가독성: st.dataframe을 활용한 데이터 그리드 최적화

8. 제약 사항
서버 구축 금지: 별도의 서버(EC2 등) 없이 무료 호스팅 환경에서만 작동해야 함.

데이터베이스: 오직 구글 시트(Google Sheets)만 사용하여 데이터를 저장하고 불러옴.

API 키 보안: 모든 API 키는 코드 내 하드코딩을 절대 금지하며, Streamlit Secrets 등의 환경 변수 기능을 사용하여 해킹을 방지함.

로그인 배제: 별도의 로그인/회원가입 기능 없이 링크 접속 시 누구나 사용 가능하게 구성함.

API 호출 제한: 각 API의 Rate Limit을 고려하여 n초/n분 단위의 효율적인 데이터 수집 로직 적용.

면책 조항: 모든 정보는 투자 참고용이며 실제 투자의 책임은 사용자에게 있음을 하단에 명시.

9. [추가 기획] RAG 기반 전천후 매크로 & 섹터 지식 자동화 엔진
- **핵심 목적**: 앱이 글로벌 매크로 뉴스와 지표를 24시간 수집/분석하여 중장기적 투자 사이클(Phase)과 자금 이동을 진단하는 '지능형 리서치 센터' 기능 추가.
- **RAG 파이프라인 (비용 최소화 구조)**: 
  - 자동 수집 (RSS 등): 주요 언론사(블룸버그, 파이낸셜타임스, 매경 등)의 실시간 뉴스를 RSS 피드와 무료 API를 통해 지속 수집.
  - 가벼운 RAG 검색: 별도의 유료 Vector DB 대신, 최신 기사 텍스트를 LLM(제미나이 등)의 대형 컨텍스트 윈도우에 실시간 주입하여 '컨텍스트 기반 최신 데이터 우선 참조' 구현.
- **매크로 사이클 분석**: 수집된 데이터를 바탕으로 현재 시장의 Phase(예: 인프라 투자기 vs. 서비스 성장기)를 진단.
- **정형화된 분석 출력 (JSON 강제)**: AI의 분석 결과는 UI 렌더링을 위해 항상 다음과 같은 JSON 형태로 반환되도록 프롬프팅 적용.
  ```json
  {
    "macro_phase": "현재 시장 매크로 사이클 진단 요약",
    "key_insight": "최신 뉴스/데이터 기반 핵심 시사점 3줄 요약",
    "bullish_sectors": ["자금 유입 수혜 섹터1", "섹터2"],
    "action_point": "구체적인 투자 스탠스 및 전략"
  }
  ```
